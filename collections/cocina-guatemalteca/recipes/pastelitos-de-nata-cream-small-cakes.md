## Ingredients

- media libra de harina
- media cucharadita de sal
- media cucharadita de Royal
- dos onzas de mantequilla
- la cuarta parte de un vaso de natas
- huevo

## Directions

1. Se pone media libra de harina, media cucharadita de sal, media cucharadita de Royal, dos onzas de mantequilla y la cuarta parte de un vaso de natas
2. Se amasa
3. Se tiende con el bolillo
4. Se cortan las figuras
5. En hojalatas se embetunan con huevo
6. Se meten al horno
